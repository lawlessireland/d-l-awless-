📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

If something doesn't work:
1. Check browser console (F12) for errors
2. Click anywhere on the page first (AudioContext requires user gesture)
3. Verify volume sliders are above 0%
4. Ensure a patch is loaded (check Play tab hero section)
5. Clear browser cache and refresh
6. Try a different browser (Chrome recommended)
7. See the Testing Guide for detailed troubleshooting

For audio glitches:
→ Increase note length if hearing clicks
→ Reduce drive if distortion is excessive
→ Lower reverb wet if audio is muddy
→ Check Audio Mix panel — ensure channels are ON
